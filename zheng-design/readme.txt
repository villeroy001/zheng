-设计模式三大类
    1.创建型模式(create):
        -普通工厂模式 factory
        -静态工厂模式 static factory
        -抽象工厂模式 abstract factory
        -单列模式 signleton
        -原型模式 prototype
        -建造者模式 builder
    2.结构型模式(structure):
        -适配器模式 adaptor
        -装饰器模式 decorator
        -代理模式 proxy
        -桥接模式 bridge
        -组合模式 composite
        -享元模式 flyweight
        -外观模式 facade
    3.行为型模式(behavior):
        -策略模式 strategy
        -模板模式 template
        -责任链 chain
        -命令 command
        -状态 state
        -备忘录 memento
        -迭代 iterator
        -观察者 observer
        -中介者 mediator
        -访问者 vistor
        -解释器 interpreter

-设计模式六大原则
    1.开闭原则: 对扩展开放，对修改关闭
    2.里氏原则: 基类与子类的继承抽象化
    3.依赖到转原则: 针对接口编程，依赖抽象而不是依赖具体
    4.接口隔离原则: 使用多个隔离接口，比单个接口更好
    5.迪米特原则: 最少知道原则，一个实体尽量少的与其他实体之前发生湘湖作用，使得模块相对独立
    6.合成复用原则: 尽量使用合成、聚合，而不是继承
